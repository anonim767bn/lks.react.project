# lks.react.project
## Описание
Проект личного кабинета сотрудника компании, оформленный с помощью библиотеки **bootstrap**
## Структура
```
lks.react.project

|    ├── components

|    │       ├── main-page

|    │       ├── profile-page  
|    │       ├── service-detail

```

## Установка
1. Клонируйте репозиторий:
2. Перейдите в директорию проекта:  
`cd lks.react.project`
3. Установите зависимости:  
`npm install`
4. Запустите:  
`npm start`
